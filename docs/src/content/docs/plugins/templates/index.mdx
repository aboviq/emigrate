A template plugin is a plugin that generates the contents of new migration files.

:::tip[Did you know?]
Many of the <Link href="/plugins/loaders/">Loader Plugins</Link> usually include a template plugin as well.
The template plugin is then responsible for generating migration files in a specific format and the loader is responsible for loading the same format.
:::

## Available Template Plugins

<CardGrid>
  <LinkCard title="JavaScript template" href="js/" description="A built-in template plugin that can generate JavaScript or TypeScript migration files (in either ESM or CommonJS format)" />
  <LinkCard title="PostgreSQL template" href="postgres/" description="A template plugin that can generate SQL, JavaScript or TypeScript migration files" />
  <LinkCard title="MySQL template" href="mysql/" description="A template plugin that can generate SQL migration files" />
</CardGrid>

:::note
You can also use the <Link href="/cli/new/#-t---template-path">`--template`</Link> option to specify a custom template file for new migrations, instead of using a plugin.
:::
